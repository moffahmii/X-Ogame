# Tic Tac Toe (X O)

A dynamic and interactive Tic Tac Toe (X O) game showcasing **advanced JavaScript skills**. 
Play against a friend or challenge the computer with multiple AI difficulty levels, including the **Minimax algorithm**.

---

## Features

- **Two Game Modes:** Play against a friend or the computer.
- **AI Opponent:** Three difficulty levels (Easy, Medium, Hard) with smart move logic.
- **Minimax Algorithm:** Optimal AI moves for the Hard level.
- **Interactive UI:** Dynamic DOM updates, hover effects, and animations.
- **Responsive Design:** Works on desktop and mobile devices.
- **Play Again & Restart:** Quickly reset the game without reloading.

---

## How to Play

1. Choose a mode: **vs Computer** or **vs Friend**.
2. Select your symbol: **X** or **O**.
3. Take turns and try to align 3 symbols in a row, column, or diagonal.
4. Play again or restart anytime.

## Tech Stack

- HTML5
- CSS3
